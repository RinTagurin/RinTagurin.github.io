1日目-1 プログラミングの基本

プログラミングとは？

「プログラミング」とは、「プログラム」を作ることです。 では、「プログラム」とは何でしょうか？

「プログラム」は「コンピューターへの命令」のことです。

今やコンピューターは身の回りにあふれています。パソコンやスマートフォンだけではありません。毎日使うような、テレビ、炊飯器、冷蔵庫、洗濯機などにもコンピューターが内蔵されています。それらのコンピューターは「プログラム」という命令に従って動いています。私たちは毎日「プログラム」のおかげで快適な生活を送っているのです。

プログラムは何ができるの？

コンピューターにも得意なこと、不得意なことがあります。それらの特徴を知って、プログラミングができると何ができるようになるか、考えましょう。

得意なこと

  • 人間より速く計算できる
  • いつでも、いつまでも動くことができる

不得意なこと

  • 「感情」を扱うこと

・・・どうですか？プログラミングでコンピューターを操ることができるようになれば、これまで面倒だったことをコンピューターに任せることができるかもしれません。さらに、自分の力ではできなかったことをコンピューターに頼むことができるかもしれませんね。

「プログラミング」とは何なのか、イメージがつかめてきましたか？では、次にプログラムのしくみを見てみましょう。

プログラムのしくみ

プログラムの進み方には3つの大原則があります。

  • 順次

    「命令が書かれた順番通りに処理が進む」

  • 反復

    「一定の条件を満たすまで処理を繰り返す」

  • 分岐

    「条件によって処理を枝分かれさせる」

これは、プログラミングする上でとても大切な考え方です。どのプログラミング言語を使っても、この考え方は共通です。

・・・言語？不思議に思った方もいるかもしれませんね。実は、プログラムはコンピューターと友達になるための「言語」なのです。

プログラミング言語

世界には日本語、英語、ロシア語と、様々な言語があります。これは「人と人」がコミュニケーションをとるためのものですね。実は、「人とコンピューター」がコミュニケーションをとるための「言語」があって、これにも様々な種類があります。次に代表的なものを挙げます。聞いたことがある言語はありますか？

  • C言語
  • Java
  • Python
  • Ruby

プログラミング言語には100以上の種類があります。それぞれに特徴があって、その言語が「得意なこと」も存在します。

今回使用するのは「Python」というプログラミング言語です。「パイソン」と読みます。では、これから学ぶ「Python」について知りましょう。

Pythonとは？

「Python」は、1991年に開発されたプログラミング言語です。マイクロソフト社やアップル社でも利用される、世界中で人気のある言語です。今話題の「AI / 人工知能」の分野でも活躍しています。

なぜPythonなのか？

プログラミングを始めるにあたって「学びやすさ」は重要です。「Hello World!」と表示するプログラムを例にとって、先ほど挙げた「C言語」「Java」「Python」「Ruby」を比較してみましょう。

  • C言語
#include <stdio.h>
void main() {
    print('Hello World!');
    return 0;
}
  • Java
public class HelloWorld {
    public static void main (String[] args){
        System.out.println("Hello World!");
    }
}
  • Python
print('Hello World!')
  • Ruby
puts "Hello World!"

いかがでしょうか。「C言語」や「Java」では5行で書いていることが、「Python」や「Ruby」ではたったの1行で終わりです。いざプログラミングを始めるぞ、と手を出してみたものの、初歩的な段階で5行ものプログラムが出てきたらどうでしょう。これではきっと、勉強がなかなか進みませんね。その点「Python」や「Ruby」は親しみやすい言語だといえます。

では「Python」vs「Ruby」ですが、実はこれ、とてもいい勝負なのです。ここで両者のプロフィールを見てみましょう。

Python

  • オランダの「グイド・ヴァンロッサム」氏が開発

Ruby

  • 日本の「まつもとゆきひろ」氏が開発

「Python」は世界中で使われている、すなわち、世界中でPythonが学ばれているということです。ですから、インターネットで検索した時の情報の量が豊富です。一方「Ruby」は日本発祥の言語ですから日本語での解説が充実しています。むむむ、互角だ。。。

ということで、私はこの勝負は「引き分け」だと思います。私のプログラミングの入口は「Ruby」でした。しかし途中から「Python」を使うようになりました。以後そのまま「Python」でプログラミングを勉強していきました。ですから改めて両者のプログラムを見比べた際、同じ内容でも慣れた「Python」の方がしっくりくるのです。（かっこの使い方とか、、、）結局ここは私の「好み」です。もちろん「Ruby」も素晴らしい言語ですので、興味をもったらぜひ勉強してみてくださいね。
